# Recursion

Recursion is a Java project that demonstrates various recursive functions. It includes functions for calculating factorial, Fibonacci numbers, checking palindromes, and cutting corners of arrays.

## Installation

To run this project, you need Java installed on your system. Clone the repository to your local machine and open it in your favorite Java IDE.
